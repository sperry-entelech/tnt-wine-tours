TNT Wine Tours Popup - Example for JTZ

This is a working example of what the popup should look like on the main TNT website.

Below you'll see the popup displayed. JTZ can use this as a reference for design and functionality.

Design Note: This popup uses the clean, white/light tntlimousine.com brand style with #CC0000 TNT Red accents. It should feel professional, not promotional or flashy.

Technical Note: The popup triggers 3 seconds after page load or on exit intent (desktop only). Uses cookies to show only once per 24 hours. On mobile, a slim bottom banner replaces the full popup.

What's New (v2 - March 2026)

  • Mobile: Phones get a slim bottom banner instead of a full-screen popup
  • Dark Theme Safe: all: initial + !important on everything — will look correct on any background
  • DO NOT remove !important declarations — they protect against style inheritance
